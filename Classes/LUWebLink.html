Overview

A LUWebLink represents a link associated with a particular location or merchant. A web link will always have a web URL (suitable to open in a web view or in Safari), but may also have a deep link URL, which links directly into another application.

When deepLinkURL is present, you should first check if the device can open the URL using [NSURL canOpenURL:], falling back to webURL if the URL cannot be opened.
